Pixel Royale

ABOUT THE PROJECT

In this project we wanted to start an indie video game company, Imagin Games, which ended up failing, but we got to have almost 100% defined the GDD of our video game for mobile platforms Pixel Royale.

The videogame was supposed to be a Battle Royale for mobile platforms with a component of randomness collecting boxes like Mario Kart and getting the loot there. You can check all the GDD for check all the videogame functionalities.

My Participation

I was the Founder of Imagin Games and during the development doing the role of Game Designer.
