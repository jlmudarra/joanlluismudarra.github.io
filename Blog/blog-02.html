My Top 5 Game Development Web Tools

20 August, 2020 Game Development Tools

Hello World!

Joan Lluis here, it’s been like 1 month since I publish my personal website and, I admit, the Blog it’s a little bit stuck. But I’m back! And I think that today's post will be so helpful for many people. Today I want to do a quick Top 5 Game Development web site tools which will help you a lot, I didn’t make that pages, but I’ve knowledge about them on my degree and on some Discord channels. I also made some projects with their help so they are 100% safe. Press start:

1. Itch.io: Free Assets

Itch.io free assets is the most powerful tool for quick game prototypes or game jams, there are more than 10.000 assets, 3D and 2D, of all the themes you can imagine. I used this website for quick ideation prototypes, it’s more cool to design with some visuals than with a greybox.

Check it Here!

2. Mixamo: Free 3D models and animations

Another powerful tool when you have to animate your 3D model is Mixamo, this tool created by Adobe allows you to import your model and animate it with more than 100 different animations. It’s so easy to animate the model and use it on Unity. Don’t worry if you don’t have any 3D model, you can download some sample 3D models from Mixamo.

Check it Here!

3. No Clip: Insane Level Design references

This tool is amazing! It’s a website where you can see real video game references about Level Design. The page allows you to navigate through more than 50 video game levels, and there you can get inspired to do your desired level. Or you can only spend your time looking at that great levels.

Check it Here!

4. Interface in Game: Best interface reference

No Clip allows you to check some video game levels…. OK, and Interface in the game allows you to see a lot of UI Designs of current successful video games. Definitely this tool is so powerful to get references and learn from great UI designs.

Check it Here!

5. Github Repository by michidk: Unity free Script Collection

And finally, but not less important, THANKS MICHAEL, this github user made an entire repository with a lot of FREE UNITY SCRIPTS, stop wasting your time programming some mechanics. You can download the script not only to use them, you can make your own script and be helped by that great examples.

Check it Here!

I hope this post help somebody. See you on the next post!
